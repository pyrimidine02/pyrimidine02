## 🩵 About Me

현장에서 좋아하는 음악을 듣고, 그 경험을 기록하고, 더 좋은 서비스로 만드는 사이클을 즐깁니다.

팬으로서 "이런 기능이 있으면 좋겠다"는 생각이 드는 순간 직접 만들기 시작하는 타입입니다.
서브컬처·밴드 음악 커뮤니티가 교차하는 지점에서 사람들에게 실제로 쓰이는 서비스를 만들고 싶습니다.

```
좋아하는 것 → 부족함을 느낌 → 직접 만듦 → 더 깊이 즐김 → 반복
```

`Spring Boot 3` · `Spring Modulith` · `PostgreSQL` · `Redis`
